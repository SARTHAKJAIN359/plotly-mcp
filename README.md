# plotly-mcp
Plotly MCP Server — Turns raw data (dicts, rows, or JSON) into interactive charts. 11 chart types (scatter, line, bar, histogram, box, violin, pie, heatmap, area, density heatmap, scatter matrix), 11 themes, and AI-powered chart-type suggestions. For chaining after SQL/semantic-search MCPs to visualize query results instantly.
